# starter-scripts

## How to use

```bash
cd /tmp
git clone https://github.com/vakshit/starter-scripts.git
cd starter-scripts
source scripts.sh
fresh_install
```

### List of individual functions

- basic_start
- install_zsh
- install_vscode
- install_ulauncher
- install_ros
- install_go
- install_node
- install_mailspring

#### Each of them can be used after sourcing the script and is in the same sequence as run in fresh_install
